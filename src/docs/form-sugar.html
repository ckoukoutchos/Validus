Form Sugar

Class providing shorthand methods for constructing <FormField> and <FormGroup>

class FormSugar

static field([ (fieldName: string), (type: string), (value?: any), (validators?: ValidatorFunction[]), (onChange: function) ]) : <FormField />

static group([ (groupName: string), (formFields: <FormField>[]), (onChange: function), (value?: any) ]) : <FormGroup />

field

Shorthand for creating a <FormField>. All other props for a <FormField> not included in the field method are set to the default value.

static field([ (fieldName: string), (type: string), (value?: any), (validators?: ValidatorFunction[]), (onChange: function) ]) : <FormField />

FormSugar.field(['name', 'text', '', [Validators.required], someFunction]);

Equivalent to:

<FormField fieldName={'name'} type={'text'} value={''} validators={[Validators.required]} onChange={someFunction} />

group

Shorthand for creating a <FormGroup>. All other props for a <FormGroup> not included in the group method are set to the default value.

static group([ (groupName: string), (formFields: <FormField>[]), (onChange: function), (value?: any) ]) : <FormGroup />

FormSugar.group(['name', [<FormField />], someFunction, '']);

Equivalent to:

<FormGroup groupName={'name'} onChange={someFunction} value={''}> <FormField /> <FormGroup />
