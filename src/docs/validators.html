Custom Validator Functions

You can easily write your own validator functions.

The function must accept at least one argument: the input value that is automatically passed to the function when the validator runs.

The function should return false if the value passes the validator; otherwise it should return an array with the first value being a string for the name of the error and the second being whatever information about the error you want returned.

Example

yourValidatorFunc(inputValue) { return (inputValue > 10) ? false : ['tooSmall', true]}

Example

yourValidatorFunc(max) { return (inputValue) => inputValue < max ? false : ['tooBig', { max: max, actual: inputValue }]}
