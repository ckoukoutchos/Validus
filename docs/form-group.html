FormGroup

React wrapper component that tracks the state of an aribratry number of FormFields

<FormGroup> Component

Required Props:

groupName: string

Optional Props:

errors: object | boolean = false

style: object = {}

value: object = {}

valid: boolean = false

onChange: function(formState: object)

onBlur: function(formState: object)

children: <FormField> | child elements

Example:

<FormGroup groupName={'login'} value={{'name': 'Tom'}} onChange={someFunction} style={{background:grey}} valid/> <FormField /> <FormGroup />
